# Overview

ThriveRoots is a modern web application that helps users understand the possible causes of hair fall through a structured assessment process.

Instead of providing generic recommendations, the platform collects user responses through an interactive questionnaire and presents a personalized report including a Hair Health Score, possible root causes, risk indicators, and treatment recommendations.

The project focuses on creating a clean, trustworthy, and intuitive user experience inspired by modern digital healthcare platforms.

# Getting Started

### Clone the repository

```bash
git clone https://github.com/Anonymous-mudit/thriveroots.git
```

### Navigate to the project

```bash
cd thriveroots
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit

```
http://localhost:3000
```
